# 微金100 · Mock 单机演示版

一个**自包含的 H5 演示版**：单个 `index.html`，**双击即可运行，无需任何环境 / 后端 / 联网**。用于快速给人演示完整产品流程（与微信小程序同一套设计、同一份 Mock 数据）。

## 一键启动（任选其一）

| 方式 | 操作 |
|---|---|
| **最简** | 直接**双击 `index.html`**，浏览器打开即用 |
| **macOS 一键** | 双击 **`启动.command`**（首次若被拦截：右键 → 打开 → 允许） |
| **本地服务器**（可选） | 在本目录 `python3 -m http.server 8080` 或 `npx serve`，浏览器访问 |

> 移动端观感最佳：浏览器开「开发者工具 → 移动设备模拟」，或直接手机浏览器打开。

## 可交互体验的流程

- **首页 / 行情**：实时金价大盘卡 + 黄金/白银/铂金切换 + 筛选 + 挂单列表
- **买家锁价**：点挂单「锁价」→ 锁价主页（数量 + 现金/转账双价试算）→ 确认弹窗 → 处理中缓冲 → 锁价成功（联系卖家）
- **我的**：保证金与额度 / 充值 / 级别与佣金 / 违约记录 / 实名认证 / 个人资料 / 收-取货地址 / 订阅金价提醒
- **订单**：销售中 / 锁价待处理 / 已完成 三板块 → 订单详情 → 确认交易完成（3 条核验）/ 平台代交接（¥100/4 步）/ 申请仲裁
- **发布现货**：须知弹窗 → 发布表单 → 发布成功

## 说明

- 纯演示：所有数据为本地 Mock；拨打 / 复制 / 充值 / 提交等操作仅弹提示，不产生真实效果。
- **正式版**是微信小程序（原生 + TypeScript），见仓库 [`frontend/`](../frontend/)，需用微信开发者工具运行。
- 业务规则 v1.0 见 [`业务规则确认结果.md`](../业务规则确认结果.md)。
