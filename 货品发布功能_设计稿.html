批注 · 货品发布表单

  • 1金属类别（黄金/白银/铂金）单选，紧凑纯文字卡（已移除冗余图标省空间），选中态茶金渐变 + 右上角对勾角标。
  • 2货品性质单选胶囊；选「旧料」联动出现折后实重黄色提示。
  • 3商品名称：商家自行输入的普通文本框，浅色 placeholder 提示（如「品牌板料 / 品牌黄金带证书」）。提示文案不含具体品牌名（用"品牌"代替，避免针对性与侵权）。数量保留两位小数，单位 g，右对齐。
  • 4出货要求整出/散出二选一。
    整出（不可切割整体卖）：① 全数量整出 锁定为总数量；② 固数量整出 输入固定份重（如 1000g/份），买家按倍数下单。例：卖家有 10 条 1000g 板料想逐条卖，选固数量 1000g。
    散出（可切割）：仅一个起批量设置，1g 起批且 ≤ 总数量，买家按 g 自由购买。
  • 5报价设置：整张卡片为一个跳转入口，点击进入价格详选页（大盘价屏④ / 一口固定价屏⑤）完成所有设置；表单内仅做纯文本总结展示，无颜色区分。
    报价同时体现现金价、转账价并标注（与发布清单一致）：大盘价+2.5 现金 / 大盘价+3.5 转账；若只选其一则只显示一个价，如 大盘价+2.5 现金 / 不支持转账。
  • 6多图上传，首图封面，可重排删除，单张 ≤10MB；至少 1 张。
  • 7悬浮主按钮（茶金渐变）；全部必填校验通过后点「确认发布」进入确认清单（屏⑦）。
    （注：卖家资料齐全性——实名 + 电话/微信 + 保证金——已在屏① 入口处优先校验，详见屏①批注④。）
  • 8等级 + 保证金 + 可发布上限 提示（数量正下方，参考买家锁价稿）：
    · 左侧 chip：「我的级别 L1-L9」（紫）+「保证金 ¥XX,XXX」（金）；
    · 中间：「可发布总数量 1 ~ X g」（由等级 × 保证金动态计算，超限红字）；
    · 底部金色虚线「补足保证金 →」点击路由跳转「我的 - 保证金充值」。
    · 保证金/级别是卖家发布数量与买家锁价数量的共同边界；demo 为 L1 + ¥0 + 0g 的受限态，不同等级/保证金结果不同。跳转目标 UI 属「我的」模块，不在本稿绘制。

批注 · 出货要求 · 散出态

  • 1散出（可切割）：分段切到「散出」后，下方只有一个「最低起批量」输入（取代整出的两个单选项）。默认 1g 起，可填写更高起批量，但最高不得超过总数量。
  • 2买家侧逻辑：在起批量 ~ 总数量区间内按 g 任意购买（如起批 50g、总量 10000g，则买家可买 50g、73.5g、200g…）。这是与整出最核心的区别——整体不可拆 vs 自由切割。
  • 3对照整出态（屏 2 主形态）：① 全数量整出 = 锁总数量整体卖；② 固数量整出 = 固定份重（200/500/1000…）按份叠加。例：10 条 1000g 板料想逐条卖 → 选固数量 1000g，买家按 1 份=1000g 倍数下单。两项二选一。

批注 · 报价设置 · 详选

  • 1分段控件：左侧「大盘价」为默认（优先）／右侧「一口固定价」（输入固定价，不随大盘波动）。
  • 2选「大盘价」展示实时大盘卡（沿用首页金价卡：奶油渐变 + 大数字 + 迷你折线 + 涨跌色）。
  • 3收款方式 = 价格行直接勾选（原独立段已删除）：勾选「加减现金价」+ 填写 = 支持现金；勾选「加减转账价」+ 填写 = 支持转账；不勾或留空 = 不接受。
    3 种最终体现：大盘价+2.5 现金 / 大盘价+3.5 转账 · 大盘价+2.5 现金 / 不支持转账 · 大盘价+3.5 转账 / 不支持现金。
  • 4最低防守价（选填）：大盘价低于防守价时按防守价显示；高于防守价时按实际大盘价 + 设定溢价报价。
  • 5符号用选择、数值用输入：现金价、转账价功能一致，均用左侧 + / − 二选一按钮决定正负，数值框只接受数字，保留 3 位小数（如 +0.123）。这样从交互上杜绝手填符号出错（全角／漏号／+−混填）。
  • 6本屏为「大盘价」模式；分段切到「一口固定价」是另一套独立界面，单独见屏⑤。
