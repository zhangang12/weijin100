## 全量确认（48 条 · 按类）

### A 交易与锁价
- **A1** 锁价后履约 4 小时 → ✅ 确认 4h；违约由守约方发起仲裁、平台判定违约责任。
- **A2** 4h 倒计时归零 → ❗ **继续计时 + 提示尽快交割，超时面临违约**。
- **A3** 报价快照有效期 → ❗ **点确认锁价即取快照**（无额外有效期窗口）。
- **A4** 并发抢锁仲裁 → ✅ 先到先得（按到达顺序）。
- **A5** 现金/转账点差来源 → ✅ 卖家发布时自定溢价。
- **A6** 卖家可设仅现金/仅转账/两者 → ✅ 确认。
- **A7** 锁价确认后不可取消、单方取消视为违约 → ✅ 确认。

### B 订单与交割
- **B1** 交易完成需买卖双方各自确认 → ✅ 确认。
- **B2** 一方确认、另一方超时 → ✅ 24h 自动完成。
- **B3** 申请仲裁后暂停 4h 倒计时、平台 2h 内致电 → ✅ 确认。
- **B4** 仲裁材料：聊天截图 ≤5 张 + 情况说明 ≤500 字（均必填）→ ✅ 确认。
- **B5** 仲裁中标识 → ✅ 在「锁价待处理」内加状态标签（不单独 tab）。
- **B6** 平台代交接 ¥100/单 → ✅ 发起方付·可协商分摊·需对方同意。
- **B7** 代交接完成口径：验货 + 买家打款即完成并释放双方保证金 → ✅ 确认。
- **B8** 转代交接后不可改回当面自交 → ✅ 确认。
- **B9** 订单号：母单 16 位 + 子单 2 位 = 18 位 → ✅ 确认。
- **B10** 卖家取消仅取消剩余库存、已锁价子单不可取消/改 → ✅ 确认。

### C 保证金与额度
- **C1** 保证金单价：金 ¥10/g、银 ¥0.5/g、铂 ¥5/g → ✅ 确认。
- **C2** 可交易额度 = 可用余额 ÷ 单价 → ✅ 确认。
- **C3** 最低充值 ¥500 → ✅ 确认。
- **C4** 退款 T+1 原路、无手续费、需无在途+无未结违约 → ✅ 确认。
- **C5** 冻结/扣罚时机 → ✅ 锁价成功即冻结 → 完成解冻 → 违约扣罚。
- **C6** 购买/发布上限 → ✅ 只看保证金额度，等级仅调佣金（不调上限）。

### D 等级与佣金
- **D1** 升级：每 10 笔成交升 1 级（买卖各计 1 笔），L1–L9 → ✅ 确认。
- **D2** 佣金费率 → ❗ 运营期免佣，费率后期随情况再定（费率表可调整）。
- **D3** 等级是否影响上限 → ✅ 只调佣金、不调上限。

### E 违约与信用
- **E1** 处罚阶梯：1/2/3/4+ 次 → 3/7/15/30 天 + 降 1/2/3/清零至 L1 → ✅ 确认。
- **E2** 违约扣款 = 违约克重 × 单价，赔付对手方 → ✅ 确认。
- **E3** 信用修复：每 30 笔无违约消 1 条；违约累计 = 近 12 个月未修复数 → ✅ 确认。
- **E4** 违约申诉窗口：判定后 24h → ✅ 确认。
- **E5** 功能限制期：可看行情/管理已有订单，不能发布/锁价 → ✅ 确认。

### F 货品发布
- **F1** 品类枚举 → ✅ 不需补充（板料 / 金条·银条·铂条 / 旧料）。
- **F2** 旧料折后实重 → ✅ 卖家自填、平台抽检。
- **F3** 出货方式三种（整出全量/整出固量/散出）→ ✅ 确认。
- **F4** 散出起批量 1g 起、≤ 总数量 → ✅ 确认。
- **F5** 发布溢价范围 → ✅ ±50 元/克。
- **F6** 最低防守价：报价 = max(大盘+溢价, 防守价) → ✅ 确认。
- **F7** 两种定价模式（大盘价+溢价 / 一口固定价，互斥）→ ✅ 确认。
- **F8** 商品图片 1–9 张、≤10MB、jpg/png、首图封面 → ✅ 确认。
- **F9** 发布合规须知 5 条 + 仅限水贝实物现货 → ✅ 确认（法务定稿）。
- **F10** 发布前置：实名 + 联系方式 + 保证金 → ✅ 确认。

### G 行情与提醒
- **G1** 大盘金价/销售价/回购价/溢价 来源 → ❗ = 后端接口数据源。
- **G2** 四价配色：红涨绿跌；销售价 > 回购价 → ✅ 确认。
- **G3** 订阅金价提醒 → ❗ 每品类 6–8 条、可下滑查看更多；涨到/跌到 + 目标价；应用推送/短信。

### H 账户与资料
- **H1** 联系方式：手机 + 微信均必填 → ✅ 确认；❗ 补充：**收/出货地址，对手方锁价后可见**。
- **H2** 实名认证：姓名 + 身份证号 + 人像面 + 国徽面，通过后锁定 → ✅ 确认。
- **H3** 微金号唯一、不可修改 → ✅ 确认。
- **H4** 收货/取货地址最多 5 个、区分类型 → ✅ 确认。
