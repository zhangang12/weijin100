首页 · 行情大厅（App 启动首屏 / 底部「首页」Tab）

用户打开 App 的第一屏。顶部奶油金实时金价大盘卡（金价 + 涨跌 + 走势 + 当日高低 / 销售价 / 回购价），下接黄金 / 白银 / 铂金品类折页切换、整出·散出·板料等筛选条，再到挂单货品列表——每条可直接「锁价」进入买家锁价流程。整体沿用设计集统一范式：奶油金渐变 + 卡片 + 红涨绿跌（现金红 / 转账蓝）。
右上角双开关：设计说明（S 键）／批注模式（A 键，点屏添加意见 → 导出 Markdown）。

· 平台权限模型 · 游客可看 + 会员可操作

首页是「行情查询 + 交易入口」一体首屏。游客（无需登录 / 注册）即可自由浏览全部行情、商家与挂单。仅当用户主动点击操作按钮（锁价 / 发布现货）时，才触发静默校验 → 直接路由跳转「我的」补全必填项，入口处不强制实名、校验无弹窗。

◆ 设计说明 · 第1屏

  • 1顶部品牌头：金色品牌徽标 + 「微金100 / 天天实时行情」。右侧「订阅」（金价提醒）+「⋯」更多。游客态同样展示，不拦截浏览。
  • 2大盘金价卡：奶油金渐变，与买家锁价 / 我的余额卡同源。大号金价 + 涨跌（绿跌 / 红涨）+ 迷你走势 + 报价时间脉动点（实时跳动）。「溢价 +4.73」带「?」释义。
  • 3四价位：当日最高 / 最低 / 销售价 / 回购价。按红涨绿跌：最高·销售红、最低·回购绿；销售>回购符合「平台高卖低收」。配色待确认，可一键互换。
  • 4快捷入口（可选）：金价走势 / 回收计算器，弱化为白底小卡，不与列表抢视觉。
  • 5品类折页 Tab：黄金 / 白银 / 铂金。选中态为金色折角折页（贴金店质感），切换联动金价卡与列表主题色。
  • 6筛选条：全部 / 整出 / 散出 / 板料 / 金条 / 旧料。单选，可横向滚动；选中为深色实心 Pill。
  • 7列表表头：名称·类别 / 库存·起批 / 价格(元/克) / 操作。类别（板料）以标签形式并入名称行第二行，节省 320px 宽度，对齐列宽。
  • 8挂单行：头像 + 脱敏账号 + 等级徽标（按 L 分层变色：高 L7-9 金、中 L4-6 青、低 L1-3 灰）；「融通足金价」为报价基准名。右侧库存克重下方显示商家起批要求（散出可小额起批，如 1g / 50g / 100g 起批；整出 = 全量起批，起批量 = 全部库存，如 1000g 起批不可拆）；再右为 现金现价/转账价双价（+涨跌）、「锁价」按钮 → 进买家锁价稿（游客点按先静默校验）。

◆ 设计说明 · 第2屏

  • 1品类主题色：白银切换后，大盘卡渐变转为冷调银白，走势线 / 选中折页 / 类别标签同步切换为银青色系；铂金则为青绿系。金价卡结构不变，仅换色与文案（实时银价）。

◆ 设计说明 · 第4屏

  • 1品类选择：弹窗内可独立切换黄金 / 白银 / 铂金，与首页折页解耦——可订阅非当前浏览品类。

  • 3目标价：金色下划线输入，下方快捷按当前价 ±%（+1% 已选）自动填充，亦可手动。
  • 4提醒方式：应用推送（默认开）+ 短信（默认关，复用「我的」已验证手机号）。订阅为游客可用的轻能力，不强制登录。
